1.2. Вкладка "Данные"

Интерфейс вкладки "Данные"
Данная вкладка отображает матрицу инцидентности и смежности постоенного графа, а так же отображается результат выполнения алгоритма поиска в ширину.

Матрица инцидентности

Вывод матрицы инцидентности графа.
Матрица инцидентности отображает связь ребр и вершин построенного графа.
 
Пользователь может только просматривать данные, все изменения проводятся на вкладке Граф.
Наверх

Матрица смежности

Вывод матрицы смежности графа.
Матрица смежности указывает с какими вершинами вершина имеет связь.
При ориентированном графе указывается вес ребра графа.
 
Размерность матрицы прямопропорционален количеству вершин графа.
 
Данная матрица используется для выполнения алгоритма поиска в ширину.
 
Пользователь может только просматривать данные, все изменения проводятся на вкладке Граф.
Наверх

Окно алгоритма

Окно выполнения алгоритма поиска в ширину.
 
Для выполнения алгоритма поиска в ширину необходимо выбрать вершину и нажать кнопку Выполнить.
 
Если путь к вершине найден он будет отображен в списке.
Если пути к вершине не существует, то в списке появится номер вершины и надпись Пути к вершине не существует.
 
Для вывода пути на графе необходимо дважды нажать на необходимый путь. После нажатия откроется вкладка Граф и на графе будет отображен выбраный путь.
Наверх

Выбор вершины

Выбор вершины для выполнения алгоритма поиска в ширину.
Наверх

Кнопка выполнить

Выполняет алгоритм с заданной вершиной.
Наверх

Лист путей

Результат выполнения алгоритма поиска в ширину.
При помощи двойного нажатия по необходимому пути будет совершен переход на вкладку Граф и показан выбранный маршрут.
